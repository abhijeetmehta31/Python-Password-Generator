# 🔐 Python Password Generator

A simple terminal-based Password Generator built using Python. It generates strong random passwords, stores password history in a text file, and allows users to view or clear saved passwords.

## 🚀 Features

- Generate random passwords
- Custom password length
- Uses uppercase letters
- Uses lowercase letters
- Uses numbers
- Uses special characters
- Saves password history
- View saved passwords
- Clear password history
- File handling
- Exception handling

## 🛠️ Technologies Used

- Python 3
- random module
- string module
- File Handling

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Python-Password-Generator.git
```

2. Navigate to the project folder

```bash
cd Python-Password-Generator
```

3. Run the program

```bash
python password-generator.py
```
